# Pattern-Numbers-Stars---1

Take as input N, a number. Print the pattern as given in output section for corresponding input.

Input Format:
Enter value of N

Constraints:
Output Format
All numbers and stars are Space separated

Sample Input
5

Sample Output
1 2 3 4 5
1 2 3 4 * 
1 2 3 * * *
1 2 * * * * *
1 * * * * * * *
